# 优质资源站点参考

以下是经过验证的优质学习资源站点。搜索时应优先从这些平台查找资源。

---

## 一、编程与开发类

| 网站 | 特点 | 适合谁 |
|------|------|--------|
| **freeCodeCamp** | 完全免费，项目驱动，2026年更新了数据库和全栈课程 | 零基础转行、需要完整学习路径的人 |
| **The Odin Project** | 强制使用本地工具（Git、终端、VS Code），贴近真实开发流程 | 自学能力强、想体验真实工作流的人 |
| **Codecademy** | 浏览器内交互式学习，即时反馈 | 完全零基础、需要手把手引导的人 |
| **Exercism** | 83种语言、8500+练习，含免费导师代码审查 | 已入门、需要大量练习和反馈的人 |
| **MDN Web Docs** | Web 标准权威文档，含学习路径和测验 | 前端开发者日常查文档、验证 API |
| **GeeksforGeeks** | 数据结构与算法、语言专题深度教程 | 刷算法、准备面试、深入某门语言 |
| **javascript.info** | 现代 JavaScript 最全面的免费教程 | 前端开发者系统学 JS |
| **Coursera / edX** | 名校课程（CS50、MIT 等），可旁听免费 | 想打牢计算机科学基础的人 |
| **LeetCode** | 算法题库 + 讨论区 | 刷题、准备技术面试 |

## 搜索策略

按用户目标选择优先平台：

- **编程开发** → 优先搜索 freeCodeCamp、MDN、GitHub、GeeksforGeeks
- **设计创意** → 优先搜索 UiiiUiii、doyoudo、虎课网、站酷
- **视频剪辑** → 优先搜索 Bilibili、新片场、我要自学网
- **学术考试** → 优先搜索 中国大学MOOC、网易公开课、知网、Bilibili
- **职业考证（软考/建造师/会计等）** → 优先搜索 中国人事考试网、中国教育考试网、233网校、粉笔题库、环球网校
- **语言学习（英语/日语/韩语等）** → 优先搜索 每日英语听力、TED、多邻国、BBC Learning English、沪江网校
- **公职考试（国考/省考/事业单位）** → 优先搜索 国家公务员局、公考雷达、粉笔题库、QZZN论坛
