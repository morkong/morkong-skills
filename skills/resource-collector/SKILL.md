# 在线学习资源全网搜集整理

这个技能定义了一套完整的"全网学习资源搜集、验证、整理"流程，适用于任何领域的备考或学习资源收集。

## 执行流程

### 阶段一：多维度搜索（4-6 路并行）

根据目标主题，同时从以下维度发起搜索，确保覆盖面：

| 维度 | 搜索词模板 | 目标内容类型 |
|------|-----------|-------------|
| 视频教程 | `{主题} 免费视频教程 B站 2024 2025` | B站/慕课网/YouTube 等平台的视频课程 |
| 书籍教材 | `{主题} 教材 推荐书籍 教程 备考` | 官方教材、辅导书、电子版PDF |
| 笔记博客 | `{主题} 考点总结 笔记 博客 CSDN 知乎` | 文字类考点精华、备考经验 |
| 在线题库 | `{主题} 真题 在线题库 刷题网站 免费` | 刷题平台、App、小程序 |
| 官方资源 | `{主题} 官网 考试大纲 报名` | 官方网站、大纲、政策文件 |
| 社区讨论 | `{主题} 备考经验 资源推荐` | 知乎/贴吧/V2EX/LINUX DO 等经验帖 |

### 阶段二：补充收网（1-2 轮）

根据初轮搜索发现的优质线索，进行定向深挖：

1. **UP主/作者深挖**：如果发现某个UP主（如B站搜索中出现的优质创作者），用 `site:平台域名 {UP主名} {主题}` 搜索其全部相关作品
2. **电子资源搜索**：搜索 `{主题} 免费电子版PDF 下载` 找到网盘分享
3. **平台确认**：对发现的每个刷题/课程平台，单独搜索确认其免费/付费策略

### 阶段三：链接验证（3-5 个关键链接）

对收集到的重要链接，用 WebFetch 工具验证：
- 页面是否真实存在（而非404）
- 页面内容是否与主题相关
- 免费/付费状态确认

### 阶段四：结构化整理

将验证后的资源按类别整理为 Markdown 表格，每条资源包含以下字段：

| 字段 | 说明 |
|------|------|
| **标题** | 资源名称，要求精炼且能一眼看出来源平台 |
| **描述** | 1-2句话说明资源内容和特色亮点 |
| **链接** | Markdown 格式的可点击链接 `[平台名](URL)` |
| **推荐值** | ⭐⭐⭐⭐⭐（五星制），免费且内容全面者给五星 |
| **是否付费** | 免费 / 付费（约¥XX）/ 部分免费 |
| **学习进度** | 默认留空，供用户自行填写 |

### 阶段五：输出攻略

在所有资源表格之后，附加一份 **备考建议总结**：

1. **核心三件套**：从免费资源中精选3条最核心的，构成零成本备考方案
2. **考试/学习概览**：该考试的科目/题型/时间/通过规则等关键信息
3. **分阶段计划**：打基础 → 强化突破 → 冲刺模拟 三阶段建议

## 关键原则

1. **免费优先**：重点发掘和推荐免费资源，付费资源标注清楚价格
2. **链接可点击**：所有链接使用 Markdown 格式，确保在阅读器中可直接点击
3. **去重合并**：同一资源出现在多处时只保留一条记录
4. **时效性标注**：视频/文章标注发表年份，让用户判断时效性
5. **验证优先**：重要链接必须经过 WebFetch 验证，避免死链
6. **搜索词必须与主题相关**：每次使用时根据用户指定的主题替换搜索关键词
